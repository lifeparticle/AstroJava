# AstroJava [development in progress]

AstroJava is a Java library for Android which provides access to Sunrise, Sunset, Moonrise, Moonset information.

To access the data for Sun
```java
Sun sun = new Sun();
sun.getSunrise();
sun.getSunset();
```

To access the data for Moon
```java
Moon moon = new Moon();
moon.getSunrise();
moon.getSunset();
```
